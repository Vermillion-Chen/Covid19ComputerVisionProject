About

This webapp was created by Victor Chen for the Fall 2021 semester of MAIS 202. It is a web application that uses convolution neural networks to detect whether or not COVID 19 is present in an x-ray scan. Based on Python and utilizes the Tensorflow library created by Google. The accuracy of the model is around 85%, and the dateset used to train the convolutional neural network is available here.
The GitHub Repository is available here.


The dataset used to train the convolutional neural network has been provided by:
  • M.E.H. Chowdhury, T. Rahman, A. Khandakar, R. Mazhar, M.A. Kadir, Z.B. Mahbub, K.R. Islam, M.S. Khan, A. Iqbal, N. Al-Emadi, M.B.I. Reaz, M. T. Islam, “Can AI help in screening Viral and COVID-19 pneumonia?” IEEE Access, Vol. 8, 2020, pp. 132665 - 132676.
  • Rahman, T., Khandakar, A., Qiblawey, Y., Tahir, A., Kiranyaz, S., Kashem, S.B.A., Islam, M.T., Maadeed, S.A., Zughaier, S.M., Khan, M.S. and Chowdhury, M.E., 2020. Exploring the Effect of Image Enhancement Techniques on COVID-19 Detection using Chest X-ray Images. arXiv preprint arXiv:2012.02238.
